Linux CLI Notes

A personal cheat sheet for Linux command line essentials — organised by topic with commands, descriptions, and examples. Updated as I progress through The Linux Command Line.

Navigation

  • cd – change directory
cd ..         # up one directory
cd /etc       # go to /etc
cd /usr/bin   # go to /usr/bin
cd ~          # current user's home
cd ~$USER     # specified user's home (if readable)
  • ls – list directories/files
ls -l     # long list
ls -la    # include hidden files
ls -li    # show inode numbers
  • pwd – print working directory
pwd
  • ALT/CTRL/A-z – alt and ctrl key shortcuts
CTRL-A         # Move cursor to the beginning of the line
CTRL-E         # Move cursor to the end of the line
CTRL-F/→       # Move cursor forward one character; same as right → key
CTRL-B/←       # Move cursor back one character; same as left ← key
ALT-F          # Move cursor forward one word
ALT-B          # Move cursor back one word
CTRL-L         # Clear the screen and move the cursor to the top-left
                              corner. Same as clear command.
CTRL-D         # Delete the character at the cursor location 
CTRL-T         # Transpose the character at the cursor location with 
                              the one preceeding it.
ALT-T          # Transpose the word at the cursor location with the 
                              word preceeding it.
ALT-U          # Convert the characters from the cursor location 
                              to the end of the word to uppercase.
CTRL-K         # Kill text from cursor location to the end of the line
CTRL-U         # Kill text from cursor to beginning of the line
ALT-D          # Kill text from cursor location to end of current word
ALT-Backspace  # Kill text from cursor location to the beginning 
                              of the current word. If the cursor is at the 
                              beginning of a word, kill the previous word.
CTRL-Y         # Yank text from the kill-ring and insert it at the
                              cursor location.

File Control & Manipulation

  • mkdir – make new directory
mkdir NAME           # create directory
mkdir -p dir/newdir  # create nested dirs
  • touch – create new empty file
touch FILE           # create empty file or update mtime
  • rm – remove files/directories
rm FILE              # remove file
rm -r DIR            # remove directory recursively (careful)
  • rmdir – remove empty directory
rmdir DIR            # remove empty directory
  • nano – edit file (creates if missing)
nano FILE            # open in nano (creates if missing)
  • cp – copy files/directories
cp SRC DEST          # copy file
cp -r SRCDIR DEST    # copy directory recursively
  • mv – move/rename
mv OLD NEW           # move/rename (does NOT edit files)

Searching

  • find – search filesystem by name/type
find PATH -name "pattern"           # by name
find PATH -type f -name "*.log"     # only files
find PATH -type d -name "conf*"     # only directories
  • grep – search text in files
grep "text" FILE
grep -i "text" FILE                 # ignore case
grep -r "text" DIR                  # recurse directory

Linking

  • ln – create hard/symbolic links
ln TARGET LINKNAME             # hard link
ln -s /path/target symlink     # symbolic link

Output & Redirection

  • redirection – send stdout/stderr to files
cmd > out.txt             # stdout to file (overwrite)
cmd >> out.txt            # stdout append
cmd 2> err.txt            # stderr to file (overwrite)
cmd 2>> err.txt           # stderr append
cmd &> all.txt            # stdout+stderr (bash)
# portable equivalent:
cmd > all.txt 2>&1
  • pipe – chain commands
cmd1 | cmd2               # pipe cmd1 stdout into cmd2
  • tee – write to file and pass through
cmd | tee out.txt | cmd2
cmd | tee -a out.txt      # append

Help & Documentation

  • man – open manual
man CMD
  • whatis – one-line description
whatis CMD
  • man sections – section numbers
  • 1 — user cmds
  • 2 — system calls
  • 3 — C library
  • 4 — special files
  • 5 — file formats
  • 6 — games
  • 7 — misc

Discovery and Exploration

  • nmap – network exploration & port scanner
nmap -sO              # IP protocol scan: which protocols host supports
nmap -sn              # Ping scan: host discovery only (skip port scan)
nmap -sL              # List scan (no packets to targets)
nmap -sT              # TCP connect(): full 3-way handshake
nmap -sS              # TCP SYN (half-open) scan
nmap -sU              # UDP scan
nmap -sV              # Service version detection
nmap -F               # Fast scan (top 100 ports)
nmap -p80,443         # specific ports
nmap -p1-1024         # port range
nmap -p-              # all ports
nmap -O               # OS detection
nmap -A               # Aggressive: OS/version/scripts/traceroute
nmap -Pn              # Treat all hosts as online; skip host discovery
nmap -T<0-5>         # Timing: paranoid(0)…insane(5)
nmap -v                # verbose (repeat v up to -vvvv)
nmap -d                # debug   (up to -d9)
nmap -oN scan.txt      # normal output
nmap -oX scan.xml      # XML output
nmap -oG scan.grep     # grepable output
nmap -oA prefix        # all major formats with a prefix

# performance tuning
nmap --min-parallelism <num>  --max-parallelism <num>
nmap --min-rate <pps>         --max-rate <pps>
nmap --host-timeout <time>
